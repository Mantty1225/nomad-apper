# Nomad Apper - 数字游民应用推荐系统

这是一个为数字游民提供专业应用推荐的Streamlit应用。

## 功能特点

### 基础分析（免费）
- 根据职业推荐相关应用
- 基础应用列表和评分
- 简单的使用建议

### 深度付费分析（¥99）
- **完整应用推荐**：5个类别的所有专业应用
- **工作流程建议**：详细的实施步骤和时间规划
- **成本分析**：详细的预算规划和节省建议
- **学习路径**：分阶段的掌握计划
- **个性化建议**：基于用户画像的定制化方案

## 快速开始

### 本地运行
```bash
git clone <repository-url>
cd nomad-apper
pip install -r requirements.txt
streamlit run app.py
```

### 部署到Streamlit Cloud
1. 连接GitHub仓库到Streamlit Cloud
2. 自动部署

## 技术栈
- Streamlit - Web框架
- Pandas - 数据处理
- NumPy - 数值计算

## 使用流程

1. 填写个人信息（姓名、邮箱、职业等）
2. 获取免费的基础分析
3. 选择是否购买深度付费分析
4. 获取完整的个性化建议

## 付费功能
深度分析包含更多专业化的建议，包括：
- 完整的应用生态系统
- 详细的工作流程实施方案
- 个性化的成本控制策略
- 持续的学习和支持
